YOLOv12 Training

Train a custom YOLOv12 detector on your own dataset. The ultralytics package handles data loading, augmentation, training loop, and checkpoint export. Below is the canonical recipe straight from the Python API. 

from ultralytics import YOLO

# Start from a pretrained checkpoint (recommended).
model = YOLO("yolo12n.pt")

# Train on a YAML dataset spec.
results = model.train(
    data="coco8.yaml",   # path to your dataset YAML
    epochs=100,
    imgsz=640,
    batch=16,
    device=0,            # GPU index, "cpu", or "mps"
)

# Evaluate on the held-out val split.
metrics = model.val()

# Export to TensorRT for ~2× faster inference on Jetson.
model.export(format="engine")
